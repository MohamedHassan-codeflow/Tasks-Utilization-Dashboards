# 🚀 Task Distribution & Utilization Dashboards

## 📌 Overview
This project represents a **suite of Excel and Power BI-based tools** designed to manage task distribution and monitor team utilization across multiple accounts.

These tools help track incoming tasks, ensure fair workload distribution among team members, and provide clear daily insights into team performance and efficiency.

---

## ⚡ Key Features
- 📥 Track tasks ready for assignment from multiple accounts  
- ⚖️ Distribute tasks evenly among **20 team members**  
- 📊 Generate daily utilization dashboards per team member and account  
- 📈 Visualize total processed tasks across different accounts  
- 🧠 Improve workload balancing and decision-making  

---

## 🛠️ Technologies Used
- 📊 Advanced Excel (Functions, Pivot Tables, Automation)  
- 📉 Power BI (Data Visualization & Dashboards)  
- ⚙️ Data Processing & Reporting Techniques  

---

## 🎯 Impact
- 🚫 Eliminates backlog and pending tasks across accounts  
- ⚖️ Ensures fair workload distribution within the team  
- 📊 Provides clear visibility of team performance  
- 🚀 Enhances productivity and operational efficiency
